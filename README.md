# 🚀 Express CRUD Application

A simple **CRUD (Create, Read, Update, Delete) Web Application** built using **Node.js and Express.js**.  
This project demonstrates how to perform basic CRUD operations in a backend application using Express.

It is a beginner-friendly project that helps understand how backend servers handle data creation, retrieval, updating, and deletion.

---

# 📌 Features

✔ Create new data  
✔ View all records  
✔ Update existing data  
✔ Delete records  
✔ Simple backend CRUD logic  
✔ Beginner-friendly project structure

# ⚙️ Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```
git clone https://github.com/harshiii7/Express-CRUD.git
```

### 2️⃣ Navigate to project folder

```
cd Express-CRUD
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the server

```
node index.js
```

---

# 💻 Usage

After starting the server, open your browser and go to:

```
http://localhost:3000
```

You can now perform CRUD operations like:

- Creating new records
- Viewing existing records
- Updating records
- Deleting records
